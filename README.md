# Mazowieckie Rycerstwo

Strona poświęcona historii polskiego oręża oraz osiągnięciom polskich skoczków narciarskich. Dostępna pod adresem: **[svaigas.github.io/mazowieckierycerstwo](https://svaigas.github.io/mazowieckierycerstwo)**

## Zawartość

### Bitwy Polskie
Opisy najważniejszych bitew w historii Polski:
- Bitwa pod Grunwaldem (1410)
- Bitwa pod Chocimiem (1621)
- Odsiecz Wiedeńska (1683)
- Bitwa pod Kircholmem (1605)
- Bitwa pod Racławicami (1794)
- Bitwa Warszawska — Cud nad Wisłą (1920)
- Monte Cassino (1944)
- Obrona Westerplatte (1939)

### Skoki Narciarskie
40 lat polskich sukcesów w skokach narciarskich:
- Profile zawodników: Adam Małysz, Kamil Stoch, Piotr Żyła, Dawid Kubacki, Stefan Hula, Andrzej Stękała
- Tabela rekordowych skoków
- Polskie złote medale olimpijskie

## Technologie

Czyste HTML/CSS/JavaScript — brak zewnętrznych zależności, brak build stepu.

## Uruchomienie lokalnie

```bash
git clone https://github.com/Svaigas/mazowieckierycerstwo.git
cd mazowieckierycerstwo
open index.html        # macOS
# lub: xdg-open index.html  (Linux)
# lub: start index.html     (Windows)
```

## Deployment

Strona jest hostowana na GitHub Pages z brancha `master`. Każdy push do `master` automatycznie aktualizuje stronę.
